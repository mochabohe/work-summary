常见问题

我的代码和文档会被上传吗?⌄

不会。项目扫描、Git 分析均在本地完成,只有最终生成总结时,才会按你的配置把必要的摘要发送给你选择的大模型服务商。

支持哪些大模型?⌄

内置 DeepSeek / Claude / OpenAI 兼容接口,也可在设置中添加自定义模型 Profile 并快速切换。

是否支持 macOS / Linux?⌄

已支持 Windows 和 macOS(Intel / Apple Silicon 通用包)。首次在 Mac 上打开时,由于未做 Apple 代码签名,需右键 → "打开" → 同意 一次即可。Linux 版本暂未发布,有需求可提 Issue。
